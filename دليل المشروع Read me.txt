نظرة عامة
نظام المدرسة هو نظام سطح مكتب تم بناءه لتننظيم العملية التعليمية في المدرسة وتخزين بيانات 
الطلاب ودرجاتهم والمقررات الدراسية  وكذالك الكادر التعليمي

هيكلية النظام
النظام مبني بلغة ال C# and SQL Server and Dev Express
وقد تم في بناءه تطبيق المفاهيم البرمجية الأساسية مثل 
OOP
Data Structure
وأيضا تم استخدام نمط
MVP Model View Presenter design pattern
وهو من الأنماط التي تجعل الكود البرمجي سهل القراءة والتعديل والاضافة وكذالك تجعله اكثر مرونة في الصيانة والتطوير

الخدمات التي يقدمها النظام
1- تخزين بيانات الطلاب ودرجاتهم 
2- تخزين بيانات المقررات الدراسية لكل صف
3- تخزين بيانات الكادر التعليمي (المعلم)
4- امكانية اضافة طالب ومعلم 
5- امكانية تسجيل سنة دراسية جديدة واضافة كل المعلومات الخاصة بها 
6- امكانية تاكيد نجاح الطالب  الى المرحة الدراسية التالية دون الحاجة الى اعادة كتابة بيانات الطالب من جديد  
7- امكانية طباعة تقرير لكل طالب بدرجاته لكل ترم بمجرد انتهاء عملية إدخال الدرجات
